Charlotte Theater : 개인

이 페이지는 프로젝트 개요, 사용한 도구, 라이브링크 및 프로젝트를 통해 배운 점들로 구성되어 있습니다.

 GitHub Repository

프로젝트 개요

기존에 있는 극장 웹사이트를 반응형으로 리뉴얼하였습니다. HTML, CSS 공부를 한 후, JavaScript와 SCSS 문법을 공부하면서 작업한 저의 "두번째 프로젝트"이며 기간은 2022.10-2022.12로 3개월간 작업하였습니다. 직접 기획, 디자인, 코딩 모두 작업하였습니다. 또한 CSS 사용 시, 반복작업의 번거로움을 개선하고자 SCSS를 활용하여 작업 속도를 1개월 단축하였습니다.

사용한 도구

HTML
SCSS
JavaScript
Vercel
Swiper

개발 포인트

  • - 첫번째 프로젝트인 분당우리교회에 비해 CSS 가독성 향상 (SCSS 사용)
  • - 모바일 우선, 반응형 UI 개발 (모바일, 태블릿, PC)
  • - 반응형 분기에 따라 다른 슬라이더 스타일 적용 (모바일: 카드UI / 태블릿 및 PC: 일반적인 슬라이더 UI)
